# 创意发散 - 项目复刻指南

## 项目简介

一个基于 AI 联想的创意发散工具。输入一个词，AI 自动联想 7-8 个相关词，以可交互的节点图谱展示，用户可以选择多个词语让 AI 生成创意方案。

**技术栈：** Vite + 原生 JS（前端）| Express + DeepSeek API（后端）

---

将以下内容发给 Claude Code，即可复刻本项目：

---

```
我需要制作一个"创意发散"的 Web 应用。功能如下：

## 核心功能
1. 用户输入一个词，后端调用 DeepSeek API 进行发散联想，返回 7-8 个中英文词对
2. 以可交互的节点图谱展示联想词（圆形玻璃态节点 + 曲线连线）
3. 点击节点可继续向下联想（展开子节点），支持折叠/展开，显示子节点数量徽章
4. 右键选择节点，选中后可让 AI 基于选中词语生成创意方案（弹窗展示）
5. 历史记录功能（localStorage 存储，侧边抽屉展示，支持恢复和删除）

## 画布交互
6. 画布平移缩放：空白处拖拽平移，鼠标滚轮缩放（向光标方向，范围 0.2x-5x）
7. 节点可自由拖拽，缩放后拖拽坐标必须正确（通过 screenToWorld 坐标转换）
8. 左下角画布控件：放大、缩小、适应视图、清空画布，显示当前缩放百分比

## 视觉设计
9. 毛玻璃（glassmorphism）设计风格，backdrop-filter + 半透明背景
10. 黄色（#FFD600）作为强调色，白色/透明玻璃背景
11. 暗色模式：CSS 变量方案，切换按钮在左上角，localStorage 持久化
12. 新节点入场动画（缩放 + 淡入），节点浮动游动动画
13. 连线使用 SVG 贝塞尔曲线（不用直线）
14. 空白画布显示欢迎状态提示

## 技术要求
15. 前端：Vite + 原生 JS（不用框架），单 CSS 文件
16. 后端：Express 代理，调用 DeepSeek chat API（deepseek-chat 模型）
17. API Key 放 .env 文件，gitignore 排除
18. Vite 开发代理 /api → localhost:3001
19. DeepSeek 返回的 JSON 需要提取 [...] 子串再解析（兼容 markdown 包裹）
20. 响应式设计（<600px 媒体查询适配移动端）

## 文件结构
- src/main.js — 应用入口，初始化所有模块
- src/graph.js — 节点图谱核心（平移缩放、拖拽、折叠、曲线连线）
- src/input.js — 输入框组件
- src/history.js — 历史记录（localStorage）
- src/generator.js — 创意生成按钮 + 结果弹窗
- src/api.js — fetch 请求封装
- src/style.css — 全部样式
- server/index.js — Express 后端，DeepSeek API 调用
```
