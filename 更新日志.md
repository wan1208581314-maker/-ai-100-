# 更新日志

## 2026-05-08 第二次更新

### 1. 点击节点展开联想（去掉弹窗）
- 节点点击改为直接展开联想词，不再需要通过弹窗确认
- 区分拖拽和点击：只有鼠标实际移动才算拖拽，避免拖拽误触发展开
- 修复 `dragNode` 的 50ms 延迟导致点击被吞的问题

### 2. 确认加号按钮
- 点击节点后，在圆圈正下方弹出一个可爱的金色圆形加号按钮
- 点击加号触发联想展开，点击空白处自动收起
- 同一节点再次点击可切换加号显示/隐藏
- 加号使用 CSS `::before` 伪元素 + 弹入动画

### 3. 去重过滤
- 前端：`expandNode()` 过滤掉 `zh` 已存在于图谱中的词语
- 后端：prompt 新增"避免返回已出现词语"指令
- API：`fetchAssociations(word, existing)` 支持传递已有词语列表

### 4. 防重叠排版（碰撞检测）
- 新增 `resolveCollisions()` 函数，新节点展开后自动检测与所有节点的距离
- 最小间距 130px，不足时沿径向方向推开
- 迭代 3 次收敛，确保不重叠

### 5. Ctrl+Z 撤销
- 新增 `undoStack`，每次展开记录 `{nodeIds, parentId}`
- 按 `Ctrl+Z`（Mac 用 `Cmd+Z`）撤销上一次展开
- 撤销时删除节点 DOM、数据、连线，更新父节点徽章
- 清空画布时同步清空撤销栈

### 6. 固定生成 8 个词
- 后端 prompt 从"7到8个"改为固定"8个"

### 修改的文件
- `src/graph.js` — 核心逻辑改动（确认按钮、去重、碰撞检测、撤销）
- `src/style.css` — 新增确认加号按钮样式，移除 `.node` 的 `overflow: hidden`
- `src/api.js` — `fetchAssociations` 支持 `existing` 参数
- `server/index.js` — prompt 固定 8 个词 + 去重指令
