# 创意发散 - 项目复刻指南

## 项目简介

一个基于 AI 联想的创意发散工具。输入一个词，AI 自动联想 8 个相关词，以可交互的节点图谱展示，用户可以选择多个词语让 AI 生成创意方案。

**技术栈：** Vite + 原生 JS（前端）| Express + DeepSeek API（后端）

---

## 使用方法

打开 Claude Code，按顺序发送以下两段对话，即可完整复刻本项目。

---

## 第一次对话：搭建基础项目

> 将以下内容完整复制，发送给 Claude Code。

```
我需要制作一个"创意发散"的 Web 应用。功能如下：

## 核心功能
1. 用户输入一个词，后端调用 DeepSeek API 进行发散联想，返回 8 个中英文词对
2. 以可交互的节点图谱展示联想词（圆形玻璃态节点 + 曲线连线）
3. 点击节点后显示确认加号按钮，点击加号展开联想词（展开子节点），支持折叠/展开，显示子节点数量徽章
4. 右键选择节点，选中后可让 AI 基于选中词语生成创意方案（弹窗展示）
5. 历史记录功能（localStorage 存储，侧边抽屉展示，支持恢复和删除）

## 画布交互
6. 画布平移缩放：空白处拖拽平移，鼠标滚轮缩放（向光标方向，范围 0.2x-5x）
7. 节点可自由拖拽，缩放后拖拽坐标必须正确（通过 screenToWorld 坐标转换）
8. 左下角画布控件：放大、缩小、适应视图、清空画布，显示当前缩放百分比
9. Ctrl+Z 撤销上一次展开操作

## 视觉设计
10. 毛玻璃（glassmorphism）设计风格，backdrop-filter + 半透明背景
11. 黄色（#FFD600）作为强调色，白色/透明玻璃背景
12. 暗色模式：CSS 变量方案，切换按钮在左上角，localStorage 持久化
13. 新节点入场动画（缩放 + 淡入），节点浮动游动动画
14. 连线使用 SVG 贝塞尔曲线（不用直线）
15. 空白画布显示欢迎状态提示

## 技术要求
16. 前端：Vite + 原生 JS（不用框架），单 CSS 文件
17. 后端：Express 代理，调用 DeepSeek chat API（deepseek-chat 模型）
18. API Key 放 .env 文件，gitignore 排除
19. Vite 开发代理 /api → localhost:3001
20. DeepSeek 返回的 JSON 需要提取 [...] 子串再解析（兼容 markdown 包裹）
21. 响应式设计（<600px 媒体查询适配移动端）

## 文件结构
- src/main.js — 应用入口，初始化所有模块
- src/graph.js — 节点图谱核心（平移缩放、拖拽、折叠、曲线连线、撤销）
- src/input.js — 输入框组件
- src/history.js — 历史记录（localStorage）
- src/generator.js — 创意生成按钮 + 结果弹窗
- src/api.js — fetch 请求封装
- src/style.css — 全部样式
- server/index.js — Express 后端，DeepSeek API 调用
```

### 构建完成后

1. 在项目根目录创建 `.env` 文件，填入你的 DeepSeek API Key：
   ```
   DEEPSEEK_API_KEY=sk-xxxxxxxxxxxxxxxx
   ```
2. 安装依赖并启动：
   ```bash
   npm install
   npm run server   # 启动后端（端口 3001）
   npm run dev      # 启动前端（端口 5173）
   ```

## 第二次对话：优化 AI 联想质量

> 基础项目跑起来后，发送以下内容，让 Claude Code 优化后端的 Prompt，使联想词更有网感、创意方案更可落地。

```
请帮我优化 server/index.js 中的两处 DeepSeek Prompt，让联想质量和创意方案质量大幅提升。

## 一、联想接口（/api/associate）

System Prompt 改为：
你是一个潮流创意策划师，熟悉小红书、抖音、B站的热门内容风格。你的联想特点是：接地气、有画面感、带点网感和趣味性，像一个很会找灵感的年轻创意人在做头脑风暴。你只返回JSON数组，不返回其他内容。

User Prompt 改为：
用户输入了"${word}"，请围绕它联想8个词。
要求：
1. 联想词要是具体的事物、场景、动作、品牌、人名、作品名等，不要抽象概念
2. 要有"意外感"——不是一眼能想到的近义词，而是稍微跳一下但又能说通的联想
3. 带点网感：可以是热梗、互联网黑话、流行文化、年轻人的生活方式
4. 覆盖不同方向：可以是同品类、使用场景、相关人物、替代品、反面、周边、跨界混搭等
5. 每个词包含 zh 和 en，严格按JSON数组返回，不要其他文字
格式：[{"zh":"冰美式","en":"Iced Americano"},...]

temperature 设为 1.0

## 二、创意方案接口（/api/generate）

System Prompt 改为：
你是一个实战派创意策划师，产出的方案可以直接给客户提案。你擅长把天马行空的联想变成可落地的创意方案，风格受小红书爆款笔记和品牌联名案例启发。

User Prompt 改为：
用户在头脑风暴中选了这些词：${keywords}。
请生成一个创意方案，要求：
1. 给方案起一个有记忆点的标题（可以是slogan风格）
2. 一句话核心概念：用一句话说清楚这个创意"是什么"
3. 执行方案：给出3-5个具体可执行的步骤，每步要落地，不要空话
4. 适合什么平台/场景投放（小红书、抖音、线下活动、品牌联名等）
5. 目标人群是谁
6. 用中文，语气像一个资深创意总监在做提案，简洁有力，不要废话

只修改这两个接口的 prompt 和 temperature，其他代码不动。
```
